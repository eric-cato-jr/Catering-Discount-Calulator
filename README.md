# Catering Discount Calculator

This project is a C# Windows Forms application that calculates the final price of a catering contract based on the selected caterer, business type, and contract length. Discounts are applied according to predefined business rules, and the final contract price is displayed through a graphical user interface.

## Features
- Windows Forms graphical user interface
- Caterer and business selection via dropdown menus
- Contract length input in years
- Discount-based price calculation
- Button-driven calculation and output display

## How It Works
1. The user selects a caterer.
2. The user selects a business.
3. The user enters the number of contract years.
4. Clicking **Calculate** applies the appropriate discount.
5. The final contract price is displayed in the form.
